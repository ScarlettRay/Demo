# SSO  以后想把它做成点单登录的一个微框架
一个单点登录的Demo
本机上运行此demo
需修改hosts文件
添加几个域名指向127.0.0.1
例如添加如下几行 
www.A.com   127.0.0.1
www.B.com   127.0.0.1
#模拟中心认证服务器地址
www.X.com   127.0.0.1

具体执行过程

访问A.com（或B.com）,进行身份认证登录。

后台收入用户名和密码数据后请求到认证服务器上进行身份认证，
身份认证成功后，返回确定数据和证书给A.com服务器，最后给请求发放证书和cookie,
前台进入首页，js脚本进行Ajax请求(必须带证书，否则后台忽略)，
后台进行证书校验，将证书发给中心认证服务器进行校验，返回校验结果。
如果校验成功，打开B.com网页就不需登录直接进入首页
